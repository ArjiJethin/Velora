# Velora Backend

NestJS backend foundation for Velora.

## Included

- NestJS with global configuration
- Prisma wired for PostgreSQL
- Swagger available at `/api`
- Global validation, exception handling, and response wrapping
- Modular feature shells for the upcoming domains

## Scripts

- `npm run start:dev`
- `npm run build`
- `npm run lint`
- `npm run format`
- `npm run prisma:generate`
- `npm run prisma:migrate`
- `npm run prisma:studio`

## Environment

Copy `.env.example` to `.env` before running the app.
